JavaScript

What is JavaScript?

JavaScript is the programming language of HTML and the Web.
Programming makes computers do what you want them to do.
JavaScript is one of the 3 languages all web developers must learn.

JavaScript can be placed in the body and the head sections of an HTML page.
You can place any number of scripts in an HTML document.

Scripts can also be placed in external files.
External scripts are practical when the same code is used in many different web pages.
JavaScript files have the file extension .js.
o use an external script, put the name of the script file in the src (source) attribute of the script tag

JavaScript can be used for many things like games.
